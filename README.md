🏃 Run locally
Would like to run this project locally? Open terminal and follow these steps:

1. Clone the repo
   git clone https://github.com/front-end-by-sberzonskiene/55-coming-soon.git
2. Install NPM packages
   npm i
or
   npm install
3. Run the server
   npm run dev
